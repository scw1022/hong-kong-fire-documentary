# Volunteer and Community Support Information

This section contains information on volunteer services, supplies support, and community mutual aid.

Please note that some websites and groups listed below are in Chinese only. In most platforms, the use of machine translators will help.

---

## 2. Resource Platforms

### 2.1 Tai Po Fire Resources (大埔火災資源整合)

- **Website**: <https://taiporesources.vercel.app/>
- **Features**: A summary of resources, including supplies, shelters, mental support, etc.
- **Contribute**: Upload the latest support information using the "Add Resources (加入資源)" feature

---

### 2.2 Wang Fuk Court Support (大埔宏福苑協尋與救助平台（宏福苑互助）)

- **Website**: <https://taipo1126.com/>
- **Features**: A one-stop website for tracing service, help & rescue, volunteer recruitment, etc.
- **Volunteer**: <https://taipo1126.com/volunteer/tasks>

---

### 2.3 Wang Fuk Court Blaze Info (宏福苑消息)

- **Website**: <https://blaze-info.vercel.app/>
- **Features**: 
  - Live emergency updates
  - Info on shelters
  - Category of supplies in need
  - Miscellaneous support information

---

### 2.4 Wang Fuk Court Support Integration Platform

- **Website**: <https://supporttaipohk.com/> (English version available)
- **Key Features**: 
  - **Resources**: Upload support information, consolidated using AI
  - **Volunteering**: Track on-duty volunteers by location using GPS (Note: Allocation features unavailable)
  - **News Updates with Credibility Voting**
  - **Support Matching**: Submit Requests, Offering, or Comments
- **Uniqueness**: Community-driven resource integration with veracity at its core

---

### 2.5 Wang Fuk Court Emergency Resources App (大埔宏福苑緊急資源簿)

- **Website**: <https://v0-emergency-resource-app-neon.vercel.app/>

## 7. Manual

### How to locate the necessary resources?

| Need | Recommended platform |
|------|--------|
| Tracing isolated persons | Safety Check APP、Family Search、Notion summaries |
| Find shelters | Resource Map, Tai Po Fire Resources |
| Obtain supplies support | Tai Po Fire Resources, SOS Station |
| Join as Volunteer | Wang Fuk Court Support (volunteer tasks), Support Integration Platform |
| Upload support information | Support Integration Platform, HK Resources on Tai Po Fire page |
| Catching up | Telegram group, Blaze Info, News on various platforms |

## 8. Community Guidelines

- **Face-checking**: You are advised to cross-check information on various platforms, especially info on resource distribution
- **Privacy protection**: Please read the privacy policies on resource platforms before uploading
- **Latest updates**: This list is up to 2025-11-28. Usable resources may be added or changed

## 9. References & Filing

Information presented on this page are sourced from community-developed support platforms. Most of these, which are released within 48 hours after the fire outbreak, are swiftly developed by contributors from the open source community, designers, social enterprises, etc. in the hopes of coordinating the community.

Please submit supplementary updates by creating a new Issue or Pull Request.
